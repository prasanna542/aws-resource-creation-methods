# EC2 Creation using AWS Management Console

## Overview
This document describes how an EC2 instance was created using the AWS Management Console.

---

## Steps Performed

1. Logged in to AWS Management Console
2. Navigated to **EC2 → Instances → Launch Instance**
3. Selected Amazon Machine Image (AMI)
4. Chose instance type
5. Configured key pair and network settings
6. Configured security group
7. Launched the instance
