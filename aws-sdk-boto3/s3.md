# S3 Creation using AWS SDK (boto3)

## Overview

This document explains how an S3 bucket was created using Python and boto3 (AWS SDK).

---

## Method

- Used boto3 client
- Specified region
- Handled LocationConstraint for non us-east-1 region
- Generated unique bucket name using timestamp

---

## Outcome

- Bucket created programmatically
- No manual interaction required

---

## Pros

- Can be integrated into applications
- Fully programmable
- Supports dynamic logic

---

## Cons

- No infrastructure state tracking
- Not ideal for large infrastructure provisioning
